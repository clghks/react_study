## React 프로젝트 생성 하기
  1. npm init
  2. npm install --save react react-dom
  3. npm install --save-dev babel-core babel-loader babel-preset-es2015 babel-preset-react
  4. npm install --save-dev react-hot-loader webpack webpack-dev-server
